# Java Mastery

Documenting my journey learning Java, from syntax fundamentals to object-oriented design and beyond.

This repository is a living log of my progress through the Java ecosystem, tracking my growth from foundational programming concepts to building real applications.

---

## Section 1: Basics
* **Console I/O:** Reading user input with `Scanner` and formatting output with `System.out.printf()`.
* **Variables & Data Types:** Primitive types (`int`, `double`, `char`, `boolean`) and non-primitive types.
* **Type Conversion:** Explicit casting between data types and string conversions using `Integer.parseInt()`, `Double.parseDouble()`, and `String.valueOf()`.
* **Built-in Classes:** Using `Math` for mathematical operations and `Date` for date/time handling.

## Section 2: Control Flow
* **Conditional Statements:** Branching logic with `if`, `else if`, `else`, and `switch` statements.
* **Loops:** Iteration using `for`, `while`, and `do-while` loops, including nested loops for pattern generation.
* **Operators:** Arithmetic, relational, logical, and modulo operators in practical scenarios.

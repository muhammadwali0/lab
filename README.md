# Lab

A personal programming lab containing small programs, scripts, and experiments across multiple languages.

This repository serves as a long-term workspace for practicing concepts, testing ideas, and building familiarity with different tools and environments.

## Structure

Each language is organized in its own directory:

* `python/`
* `cpp/`
* ...

Each directory contains independent code files focused on specific tasks, concepts, or experiments.

## Purpose

* Practice programming across multiple languages
* Experiment with different approaches and techniques
* Build consistency through hands-on coding
* Maintain a centralized workspace for learning

## Notes

* Code is intentionally simple in some places and more complex in others, depending on the concept being explored
* Files are not always interconnected and can be treated as standalone examples
* This repository is continuously expanded over time

## Usage

Browse by language and open any file to explore the implementation.

# Bash Lab

Documenting my journey from zero to advanced Bash scripting.

This repository is a living log of my progress through Bash ecosystem. It tracks my transition from writing first scripts to building production-quality shell tools.

---

## Section 0: Setup

- **OS:** Omarchy (Arch Linux)
- **Shell:** Bash 5.3.x
- **Editor:** Neovim
- **Terminal:** Ghostty

## Section 1: Basics

- **Scripting:** Writing your first `.sh` file, the shebang line, and making scripts executable.
- **User Input:** Reading input with `read` and accepting arguments with `$1`, `$2`, etc.
- **Functions:** Defining and calling functions, local variables, and return values.
- **For Loops:** Iterating over lists, ranges, files, and command output.
- **Input / Output:** stdin, stdout, stderr, redirection (`>`, `>>`, `2>`) and pipes (`|`).
